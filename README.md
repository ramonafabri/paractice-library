# Practice Library

**EN** — Small Spring Boot REST API (Java 21) for libraries and books. Practice project: JPA, MySQL, DTOs, validation, many-to-many mapping. Run with `mvn spring-boot:run` (MySQL required, see `application.yaml`).

**HU** — Kis Spring Boot REST API (Java 21) könyvtárak és könyvek kezelésére. Gyakorlóprojekt: JPA, MySQL, DTO-k, validáció, több-a-többhöz kapcsolat. Futtatás: `mvn spring-boot:run` (MySQL kell, lásd `application.yaml`).

Példa kérések: `intro.http`
